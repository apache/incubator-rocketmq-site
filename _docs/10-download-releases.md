# Compatibility Guide
The table below shows the supported jre environment for components of each release.

| Version | Client | Broker | NameServer |
| --- | --- | --- | --- |
| 4.0.0-incubating | >=1.7 | >=1.8 | >=1.8 |
| 4.1.0-incubating | >=1.6 | >=1.8 | >=1.8 |
| 4.2.0 | >=1.6 | >=1.8 | >=1.8 |
